#1 - What is this website?

This is a multi tools website that uses Youtube's videos / subtitles to create some pretty entertaining stuff, like a Pewdiepie video search engine, a 'Higher or Lower' game with frequently said words etc..

#2 - How accurate are the results?

All the results use manual subtitles (for now), so all videos since 2017 without subtitles are not counted, to be exact the 1069/1298 are being searched. The subtitle comments are also NOT included in the search so it's mostly things that are said in the actual videos

#3 - Why did you start from 2017? Why not earlier?

No reason. for now i figured the 3 past years is a reasonable time, i will probably add older videos if i can.

#4 - Will you add more feautures?
